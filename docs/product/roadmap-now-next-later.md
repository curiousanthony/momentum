# Roadmap: Now, Next, Later

## Now

- codify the product strategy and documentation system
- establish the durable memory layer for agents and future sessions
- align the existing dashboard with the approved north-star language and user model

## Next

- add stage-aware progression so the product reflects the journey from `Explorer` to `Power User`
- add richer insight mechanics that explain improvement, not only counts
- add momentum and comeback systems that reinforce consistency without guilt
- start connecting behavior patterns to project-level value

## Later

- add advanced mastery views for high-usage users
- add stronger outcome intelligence tied to project progress and shipping signals
- add more adaptive guidance based on progression stage
- expand the product into a more strategic command layer for high-leverage builders

## Prioritization Logic

The near-term roadmap should prioritize:

1. documentation and product memory
2. insight quality
3. progression and momentum
4. outcome intelligence

This order keeps the product credible before making it more complex.
